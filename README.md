# FreeSpace

FreeSpace est une marketplace d'économie circulaire pour l'immobilier professionnel en Brabant Wallon.

## Technologies

- **React** + **Vite**
- **Firebase** (Auth, Firestore, Storage)
- **Framer Motion** (Animations)
- **Leaflet** (Cartographie)
- **Tailwind CSS** (Styling)

## Déploiement

Ce projet est prêt à être déployé sur **Vercel**.

1. Connectez votre dépôt GitHub à Vercel.
2. Vercel détectera automatiquement les paramètres de build Vite.
3. Le fichier `vercel.json` gère les redirections pour React Router.

## Développement local

```bash
npm install
npm run dev
```
